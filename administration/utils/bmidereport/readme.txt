Prerequisite

1. Install ANT 1.8.0 (copy & paste)
- Set ANT_HOME to installation directory
- Add %ANT_HOME%\bin to "PATH" variable
2. Copy BMIDE sample package
3. Change properties file if necessary

Execute to generate the report

4. Open CMD, goto BMIDE sample project, run "ANT"
5. Check the results in "out" sub folder
